# 도장깨기 · 韩语单词闪卡

一个纯静态的韩语单词闪卡网页：翻卡片 + 印章式记忆分级 + 间隔复习。

## 文件说明

- `index.html` — 页面结构、样式和全部逻辑代码。
- `vocab.json` — 单词数据。新增/修改单词只需要编辑这个文件，不用碰 `index.html`。

## 怎么加新单词 / 新单元

打开 `vocab.json`，仿照已有格式在数组里加一条，例如：

```json
{"k":"사랑","r":"sarang","m":"爱","e":"저는 가족을 사랑합니다.","t":"我爱我的家人。"}
```

字段含义：
- `k` 韩语单词
- `r` 罗马音标注
- `m` 中文释义
- `e` 韩语例句
- `t` 例句中文翻译

记得在上一条后面加英文逗号 `,`，最后一条不要加逗号。存好后刷新网页就能看到新词。

## 学习进度保存在哪里

学习进度、每日新词学习数量、"每天最多学新词多少个"的设置，都保存在**访问者自己浏览器的 localStorage** 里，不会上传到任何服务器。换浏览器或换设备进度不会同步，清除浏览器数据会清空进度。

## 本地怎么预览修改效果

因为页面用 `fetch` 读取 `vocab.json`，不能直接双击 `index.html` 打开（会因为浏览器的本地文件限制加载失败），需要用一个本地小型服务器打开，例如在这个文件夹里执行：

```
python3 -m http.server 5500
```

然后浏览器打开 `http://localhost:5500` 查看。

## 部署

这是纯静态网站（没有后端），推送到 GitHub 后可以直接用 Vercel / Netlify / GitHub Pages 等任意静态托管平台部署。
