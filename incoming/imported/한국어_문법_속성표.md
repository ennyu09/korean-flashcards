# 韩语语法速查表

卡片背面的语法分析里会反复出现一些简写标注（比如"-아서/어서""-습니다"）。
这份表把它们集中整理成表格，方便查阅。

**记号说明**：下面用 **C** 代表"辅音收尾"（词干/名词最后一个字带收尾辅音，如"먹"），
**V** 代表"元音收尾"（最后一个字没有收尾辅音，如"가"）。具体这条规则怎么运作、
有哪些例外，见最后一节的详细说明；这里先把各语法点的用法列出来方便直接查。

## 一、助词（조사）

助词接在名词后面，标记这个名词在句子里的语法角色。

### 助词详细表

<a id="p-이가"></a>

| 助词 | 作用 | C收尾例句 | V收尾例句 |
|---|---|---|---|
| **이/가** | 标记主语 | 사람**이**（人） | 친구**가**（朋友） |
| **은/는** <a id="p-은는"></a> | 标记话题/对比 | 책**은**（书） | 학교**는**（学校） |
| **을/를** <a id="p-을를"></a> | 标记宾语 | 밥**을**（饭） | 사과**를**（苹果） |
| **로/으로** <a id="p-로으로"></a> | "往……""用……" | 책**으로** | 학교**로** |
| **와/과** <a id="p-와과"></a> | "和……" | 학생**과** | 친구**와** |
| **에** <a id="p-에"></a> | "往……""在……（时间点）" | 학교**에**（不分C/V，固定用에） | 학교**에** |
| **에서** <a id="p-에서"></a> | "在……（做事的地点）" | 식당**에서**（不分C/V，固定用에서） | 식당**에서** |

（의、도、만、부터、까지、께서、한테/에게这几个不受C/V影响，固定写法，不单独列表。）

### 辨析：이/가 和 은/는 的区别

表格里看起来이/가和은/는都能翻译成"是……"，但功能不同：

| | 이/가 | 은/는 |
|---|---|---|
| 作用 | 标记主语，中性客观陈述 | 标记话题/对比 |
| 典型场合 | 第一次提到某人事物；回答"谁？""什么？" | 转换话题、做对比；陈述一个已知话题 |
| 例句 | 제**가** 학생입니다.（我是学生） | 저**는** 학생입니다.（我呢，是学生——隐含对比） |

## 二、连接语尾（연결어미）

接在动词/形容词词干后，连接前后两个短句，构成一个完整复句。

| 语尾 | 核心作用 | C/V |
|---|---|---|
| **-아서/어서** <a id="c-아서어서"></a> | ①原因"因为……所以……" ②先后"……之后……" | 看元音（见第六节） |
| **-고** <a id="c-고"></a> | "……然后……"，并列或先后 | 固定不变：먹**고**／가**고** |
| **-지만** <a id="c-지만"></a> | "虽然……但是……" | 固定不变：먹**지만**／가**지만** |
| **-면** <a id="c-면"></a> | "如果……的话" | C接-으면（먹**으면**）V接-면（가**면**） |
| **-는데** <a id="c-는데"></a> | "……，不过……" | 固定不变：먹**는데**／가**는데** |
| **-니까/으니까** <a id="c-니까"></a> | "因为……所以……" | C接-으니까（먹**으니까**）V接-니까（가**니까**） |
| **-다가** <a id="c-다가"></a> | "做……到一半，（然后）……" | 固定不变：먹**다가**／가**다가** |

## 三、终结语尾与敬语体系

韩语的敬语程度由两个独立维度决定：对听者是否尊敬，场合是否正式。两者交叉出三种常用说法：

| | 尊敬 | 不尊敬 |
|---|---|---|
| 正式场合 | **尊敬阶**（합니다体，传统语法称格式体敬语/하십시오体） | 几乎不存在这种组合 |
| 日常场合 | **泛尊阶**（해요体，传统语法称非格式体敬语） | 非敬语（**반말**） |

同一件事，三种说法：

| 敬语级别 | 例句 | 中文 |
|---|---|---|
| 尊敬阶 | 식사하셨습니**까**? | 您用餐了吗？（播音员/初次商务场合） |
| 泛尊阶 | 식사하셨**어요**? | 您吃饭了吗？（日常礼貌，最常用） |
| 非敬语 | 밥 먹었**어**? | 吃饭了吗？（好朋友/晚辈） |

拿不准用哪个时，泛尊阶（해요体）是最常用、最安全的选择，教材通常也是先教해요体、再教합니다体。

### 具体语尾对照表

| 语尾 | 敬语类型 | 句子类型 | C/V |
|---|---|---|---|
| **-습니다/-ㅂ니다** <a id="e-ㅂ니다"></a> | **尊敬阶**（합니다体） | 陈述句 | C接-습니다（먹**습니다**）V接-ㅂ니다（가**ㅂ니다**→갑니다） |
| **-습니까/-ㅂ니까** <a id="e-ㅂ니까"></a> | 尊敬阶 | 疑问句 | 同上 |
| **-읍시다/-ㅂ시다** <a id="e-ㅂ시다"></a> | 尊敬阶 | 共动句（"让我们……吧"） | 同上：먹**읍시다**／가**ㅂ시다**→갑시다 |
| **-어요/-아요** <a id="e-어요"></a> | **泛尊阶**（해요体） | 陈述/疑问/命令均可 | 看元音（见第六节）：먹**어요**／가**요** |
| **-세요/-으세요** <a id="e-세요"></a> | 泛尊阶 | 请求/命令，或疑问 | C接-으세요（앉**으세요**）V接-세요（가**세요**） |
| **-어/-아/-야** <a id="e-반말"></a> | 非敬语（반말） | 各种句子类型 | 看元音：먹**어**／가**아**→가 |

## 四、时态与语气标记

插在词干和终结语尾之间。

| 标记 | 作用 | C/V |
|---|---|---|
| **-았-/-었-/-였-** <a id="t-았었였"></a> | 过去时标记 | 看元音（见第六节）：먹**었**습니다／가**았**→갔습니다；하다固定해 |
| **-겠-** <a id="t-겠"></a> | 将来/意志，或婉转推测 | 固定不变：먹**겠**습니다／가**겠**습니다 |
| **-고 있다** <a id="t-고있다"></a> | "正在……"（进行时） | 固定不变：먹**고 있습니다**／가**고 있습니다** |
| **-아/어 있다** <a id="t-아있다"></a> | "……着"（状态持续） | 看元音：앉**아 있습니다**（앉的元音ㅏ）／서 있습니다（서다元音本身是ㅓ，接어后合并） |

## 六、C/V规则详解与特殊案例

### 判断口诀

辅音收尾（C）→ 选带"으"的写法；元音收尾（V）→ 选不带"으"的写法，直接拼上去。
动词、形容词的词干也按这个逻辑判断，比如"먹다"的词干"먹"是C，"가다"的词干"가"是V。
前面几节表格里的助词、语尾，大多数都按这条规则判断该用哪个写法。

### 特殊案例①：로/으로的ㄹ例外

前面的字虽然是C（辅音收尾），但收尾辅音正好是**ㄹ**（比如"지하철"），仍然按V收尾的写法用"**로**"，不用으로。这是唯一的例外情况。

### 特殊案例②：-아/어系语尾不看C/V，看元音

-아요/어요、-아서/어서、-았/었-这几个语尾判断依据不是C/V，而是词干最后一个字的元音：

| 元音 | 接哪一类 |
|---|---|
| **ㅏ 或 ㅗ** | 아类 |
| 其他元音 | 어类 |

| 动词/形容词 | 元音 | 接哪一类 | 例句 |
|---|---|---|---|
| 가다（去，V收尾） | ㅏ | 아类 | 가**요**／가**서**／갔 |
| 좋다（好，C收尾） | ㅗ | 아类 | 좋**아요**／좋**아서**／좋았 |
| 먹다（吃，C收尾） | ㅓ | 어类 | 먹**어요**／먹**어서**／먹었 |
| 웃다（笑，C收尾） | ㅜ | 어类 | 웃**어요**／웃**어서**／웃었 |

좋다和먹다都是C收尾，但一个接아类一个接어类——这几个语尾根本不看C/V，只看元音，跟前面几节表格里其他大部分语法点的判断逻辑不同。

### 特殊案例③：하다固定变해

所有"……하다"结尾的词（공부하다、운동하다等），不按规则推导，一律固定变成"해"：공부**해요**，不是공부하아요。
