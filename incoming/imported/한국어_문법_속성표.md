## 一、助词（조사）

助词接在名词后面，标记这个名词在句子里的语法角色（主语/宾语/方向/地点等）。

### 核心辨析：이/가 和 은/는，到底有什么不同？

这两组助词经常都翻译成"是……"，但功能完全不同：

| | 이/가（主格助词） | 은/는（主题助词） |
|---|---|---|
| **作用** | 单纯标记"谁是动作的主语"，中性客观陈述 | 标记"这句话在说的是谁/什么"，常带对比、强调意味 |
| **典型场合** | 第一次提到某人事物；回答"谁？""什么？" | 转换话题、做对比；陈述一个已知话题 |
| **例句** | 제**가** 학생입니다.（我是学生——单纯陈述事实） | 저**는** 학생입니다.（我呢，是学生——隐含对比） |

初学阶段不用死记规则，多看例句会慢慢有语感。

### 助词总表

<a id="p-이가"></a>

| 助词 | 作用 | 例句 |
|---|---|---|
| **이/가** | 主格助词，标记句子主语 | 사람**이** 많습니다.（人很多） |
| **은/는** <a id="p-은는"></a> | 主题助词，标记话题/对比 | 저**는** 학생입니다.（我是学生） |
| **을/를** <a id="p-을를"></a> | 宾格助词，标记宾语 | 밥**을** 먹습니다.（吃饭） |
| **에** <a id="p-에"></a> | 方向/时间点助词，"往……""在……（时间点）" | 학교**에** 갑니다.（去学校） |
| **에서** <a id="p-에서"></a> | 处所助词，"在……（做某事的地点）" | 식당**에서** 먹습니다.（在餐厅吃） |
| **로/으로** <a id="p-로으로"></a> | 方向/手段/材料助词，"往……""用……" | 지하철**로** 갑니다.（坐地铁去） |
| **와/과** <a id="p-와과"></a> | "和……"，共同对象 | 친구**와** 만납니다.（和朋友见面） |
| **의** | 所有格助词，"……的" | 저**의**→제（我的） |
| **도** | "也……" | 저**도** 갑니다.（我也去） |
| **만** | "只……" | 이것**만** 있습니다.（只有这个） |
| **부터** | "从……开始" | 오늘**부터**（从今天起） |
| **까지** | "到……为止" | 여기**까지**（到这里为止） |
| **께서** | 主格助词敬语形式 | 할머니**께서**（奶奶，敬语） |
| **한테/에게** | "向……，给……"（对象） | 친구**한테**（对朋友） |

## 二、连接语尾（연결어미）

接在动词/形容词词干后，连接前后两个短句，构成一个完整复句。

| 语尾 | 核心作用 | 判断方法/说明 |
|---|---|---|
| **-아서/어서** <a id="c-아서어서"></a> | ①表原因"因为……所以……" ②表先后"……之后……" | 前后两件事有时间先后关系→"之后"；前面是原因、后面是结果→"因为" |
| **-고** <a id="c-고"></a> | "……然后……"，单纯并列或先后 | 通常是"做完A再做B"，不强调因果 |
| **-지만** <a id="c-지만"></a> | "虽然……但是……" | 表转折 |
| **-면** <a id="c-면"></a> | "如果……的话" | 表假设/条件 |
| **-는데** <a id="c-는데"></a> | "……，不过……" | 表铺垫、转折或背景说明 |
| **-니까/으니까** <a id="c-니까"></a> | "因为……所以……"（比-아서语气更强调"提醒/建议"） | 常用于"因为这样，所以你应该……" |
| **-다가** <a id="c-다가"></a> | "做……到一半，（然后）……" | 表动作中途转换 |

## 三、终结语尾与敬语体系

### 大前提：韩语的"礼貌程度"是两条独立的轴

很多人以为敬语只有"敬语 vs 不敬语"一条线，其实韩语里同时存在**两个维度**：

| 维度 | 说明 |
|---|---|
| **① 对听者礼貌与否** | 尊敬：长辈、上级、陌生人、需要保持礼貌的对象；不尊敬（반말）：关系很近的朋友、晚辈、小孩 |
| **② 场合正式与否** | 正式：新闻播报、公司会议、正式演讲、军队、初次商务场合，保持距离感；日常：日常聊天、朋友间、便利店买东西，礼貌但亲切自然 |

两个维度交叉，得到韩语最常见的三种说话方式（本表统一使用**尊敬阶/泛尊阶**这套说法，括号内为对应的传统语法名称）：

| | 尊敬 | 不尊敬 |
|---|---|---|
| **正式场合** | **尊敬阶**（합니다体，传统语法称格式体敬语/하십시오체） | （几乎不存在这种组合） |
| **日常场合** | **泛尊阶**（해요体，传统语法称非格式体敬语） | 非敬语（**반말**） |

**一句话理解三者关系**：

| 说法 | 心态 |
|---|---|
| 尊敬阶（합니다体） | "我们在办公事，对您保持礼貌和距离" |
| 泛尊阶（해요体） | "我对您有礼貌，但关系轻松自然，不是在开会" |
| 반말 | "我们很熟，不需要客套" |

同一件事，三种说法对比：

| 敬语级别 | 例句 | 中文 |
|---|---|---|
| 尊敬阶 | 식사하셨습니**까**? | 您用餐了吗？（像播音员/初次商务场合） |
| 泛尊阶 | 식사하셨**어요**? | 您吃饭了吗？（日常礼貌，最常用） |
| 非敬语 | 밥 먹었**어**? | 吃饭了吗？（对好朋友/晚辈） |

**给自学者的实用建议**：拿不准用哪个的时候，**泛尊阶（해요体）永远是最安全的选择**——韩国人日常说话里泛尊阶占绝大多数。这也是为什么很多教材会先教해요体、再教합니다体。

### 具体语尾对照表

| 语尾 | 敬语类型 | 句子类型 | 备注 |
|---|---|---|---|
| **-습니다/-ㅂ니다** <a id="e-ㅂ니다"></a> | **尊敬阶**（합니다体） | 陈述句 | 最正式 |
| **-습니까/-ㅂ니까** <a id="e-ㅂ니까"></a> | 尊敬阶 | 疑问句 | |
| **-읍시다/-ㅂ시다** <a id="e-ㅂ시다"></a> | 尊敬阶 | 共动句（"让我们……吧"） | |
| **-어요/-아요** <a id="e-어요"></a> | **泛尊阶**（해요体） | 陈述/疑问/命令均可 | 同一词形，靠语调和上下文区分书面上看不出来的差异 |
| **-세요/-으세요** <a id="e-세요"></a> | 泛尊阶 | 请求/命令，或疑问 | |
| **-어/-아/-야** <a id="e-반말"></a> | 非敬语（반말） | 各种句子类型 | 同样靠语境区分 |

## 四、时态与语气标记

插在词干和终结语尾之间。

| 标记 | 作用 |
|---|---|
| **-았-/-었-/-였-** <a id="t-았었였"></a> | 过去时标记 |
| **-겠-** <a id="t-겠"></a> | 表将来/意志（"会……""要……"），或婉转推测 |
| **-고 있다** <a id="t-고있다"></a> | "正在……"（进行时） |
| **-아/어 있다** <a id="t-아있다"></a> | "……着"（状态持续，强调动作完成后的状态） |

## 五、敬语标记

表示对句子主语的尊敬。

| 标记 | 作用 |
|---|---|
| **-시-/-으시-** <a id="h-시"></a> | 插入动词/形容词词干中，表示对主语的尊敬（用于长辈、上级等） |

### 特殊敬语词（整个词替换，不是加-시-）

<a id="h-특수"></a>

| 普通形 | 敬语形 | 意思 |
|---|---|---|
| 먹다 | 드시다 / 잡수시다 | 吃 |
| 자다 | 주무시다 | 睡 |
| 있다 | 계시다 | 在 |
| 말하다 | 말씀하시다 | 说 |
