# 韩语语法速查表

卡片背面的"语法分析"里会反复出现一些简写标注（比如"-아서/어서"），
这份速查表把它们集中解释一次。看到卡片上有对应标注，回来这里查就行。

## 一、助词（조사）—— 接在名词后面，标记语法角色

### 最容易混淆的一组：이/가 和 은/는，到底有什么不同？

这两组助词经常都翻译成"是……"，但**功能完全不同**：

- **이/가（主格助词）**：单纯标记"谁是动作的主语"，是中性、客观的陈述
- **은/는（主题助词）**：标记"这句话在说的是谁/什么"，常带有**对比或强调**的意味

对比感受一下：
- 제가 학생입니다 （我是学生 —— 单纯陈述事实，比如别人问"谁是学生？"）
- 저는 학생입니다 （我呢，是学生 —— 隐含"（相对于别人/别的话题）我是这样的"）

**简单判断口诀**：
- 第一次提到某个人/事物，或者在回答"谁？""什么？"这类问题时，倾向用 **이/가**
- 转换话题、做对比，或者陈述一个"总体已知"的话题时，倾向用 **은/는**

这个区别初学阶段不用完全掌握，多看例句自然会有语感，不用死记规则。

### 助词总表

<a id="p-이가"></a>
**이/가** —— 主格助词，标记句子的主语。例：사람이 (人)

<a id="p-은는"></a>
**은/는** —— 主题助词，标记句子的话题/对比。例：저는 (我)

<a id="p-을를"></a>
**을/를** —— 宾格助词，标记宾语。例：밥을 (饭)

<a id="p-에"></a>
**에** —— 方向/时间点助词，"往……""在……（时间点）"。例：학교에 (去学校)

<a id="p-에서"></a>
**에서** —— 处所助词，"在……（做某事的地点）"。例：식당에서 (在餐厅)

<a id="p-로으로"></a>
**로/으로** —— 方向/手段/材料助词，"往……""用……"。例：지하철로 (坐地铁)

<a id="p-와과"></a>
**와/과** —— "和……"，表共同对象。例：친구와 (和朋友)

<a id="p-의"></a>
**의** —— 所有格助词，"……的"。例：저의→제 (我的)

<a id="p-도"></a>
**도** —— "也……"。例：저도 (我也)

<a id="p-만"></a>
**만** —— "只……"。例：이것만 (只有这个)

<a id="p-부터"></a>
**부터** —— "从……开始"。例：오늘부터 (从今天起)

<a id="p-까지"></a>
**까지** —— "到……为止"。例：여기까지 (到这里为止)

<a id="p-께서"></a>
**께서** —— 主格助词的敬语形式。例：할머니께서 (奶奶)

<a id="p-한테에게"></a>
**한테/에게** —— "向……，给……"（对象）。例：친구한테 (对朋友)

## 二、连接语尾（연결어미）—— 接在动词/形容词词干后，连接前后两个短句

<a id="c-아서어서"></a>
**-아서/어서** —— ①表原因"因为……所以……" ②表先后"……之后……"
判断方法：前后两件事有时间先后关系→"之后"；前面是原因、后面是结果→"因为"

<a id="c-고"></a>
**-고** —— "……然后……"，单纯并列或先后，通常就是"做完A再做B"，不强调因果

<a id="c-지만"></a>
**-지만** —— "虽然……但是……"，表转折

<a id="c-면"></a>
**-면** —— "如果……的话"，表假设/条件

<a id="c-는데"></a>
**-는데** —— "……，不过……"，表铺垫、转折或背景说明

<a id="c-니까"></a>
**-니까/으니까** —— "因为……所以……"（比-아서语气更强调"提醒/建议"），常用于"因为这样，所以你应该……"

<a id="c-다가"></a>
**-다가** —— "做……到一半，（然后）……"，表动作中途转换

## 三、终结语尾（종결어미）—— 句子最后一个词，决定敬语级别和句子类型

### 先搞懂一个大前提：韩语的"礼貌程度"其实是两条独立的轴

很多人以为敬语只有"敬语 vs 不敬语"一条线，其实韩语里同时存在**两个维度**，
两个维度组合起来才决定一句话最终听起来是什么感觉：

**维度①：对听者礼貌不礼貌（尊敬与否）**
- 尊敬：对长辈、上级、陌生人、需要保持礼貌的对象
- 不尊敬（반말）：对关系很近的朋友、晚辈、小孩

**维度②：场合正式不正式（格式与否）**
- 格式体：新闻播报、公司会议、正式演讲、军队、第一次见面的商务场合——
  说话人和听话人之间保持一定"距离感"，显得庄重、公事公办
- 非格式体：日常聊天、朋友间、便利店买东西、跟同事随口说话——
  依然礼貌，但更亲切自然，是韩国人日常生活里用得最多的说法

**两个维度交叉，就得到韩语最常见的三种说话方式：**

| | 尊敬 | 不尊敬 |
|---|---|---|
| **正式场合** | 格式体敬语（합니다体） | （几乎不存在这种组合，正式场合一定要敬语） |
| **日常场合** | 非格式体敬语（해요体） | 非敬语（반말） |

**一句话理解三者的关系**：
- 합니다体 = "我们现在是在办公事，我对您保持礼貌和距离"
- 해요体 = "我对您有礼貌，但我们关系轻松自然，不是在开会"
- 반말 = "我们很熟，不需要客套"

同一件事，用三种方式说会是这样：
- 격식체：식사하셨습니까? （您用餐了吗？—— 像播音员或者第一次见面的商务场合）
- 해요体：식사하셨어요? （您吃饭了吗？—— 日常礼貌，最常用）
- 반말：밥 먹었어? （吃饭了吗？—— 对好朋友/晚辈）

**给自学者的实用建议**：不确定用哪个的时候，**해요体永远是最安全的选择**——
韩国人日常说话里해요体占绝大多数，格式体主要出现在特定正式场合，반말只在
关系明确亲近之后才用。这也是为什么很多教材会先教해요体、再教합니다体。

### 具体语尾对照表

<a id="e-ㅂ니다"></a>
**-습니다/-ㅂ니다** —— 格式体敬语（합니다体），用于陈述句

<a id="e-ㅂ니까"></a>
**-습니까/-ㅂ니까** —— 格式体敬语，用于疑问句

<a id="e-ㅂ시다"></a>
**-읍시다/-ㅂ시다** —— 格式体敬语，用于共动句（"让我们……吧"）

<a id="e-어요"></a>
**-어요/-아요** —— 非格式体敬语（해요体），陈述句/疑问句/命令句均可，同一个词形，靠语调和上下文区分（书面上看不出来，这也是为什么해요体有时会让初学者觉得"怎么什么句子都长一样"）

<a id="e-세요"></a>
**-세요/-으세요** —— 非格式体敬语（해요体的请求/命令形式），用于请求/命令，或疑问

<a id="e-반말"></a>
**-어/-아/-야** —— 非敬语（반말），各种句子类型，同样靠语境区分

## 四、时态与语气标记 —— 插在词干和终结语尾之间

<a id="t-았었였"></a>
**-았-/-었-/-였-** —— 过去时标记

<a id="t-겠"></a>
**-겠-** —— 表将来/意志（"会……""要……"），或婉转推测

<a id="t-고있다"></a>
**-고 있다** —— "正在……"（进行时）

<a id="t-아있다"></a>
**-아/어 있다** —— "……着"（状态持续，强调动作完成后的状态）

---

## 五、敬语标记 —— 表示对句子主语的尊敬

<a id="h-시"></a>
**-시-/-으시-** —— 插入动词/形容词词干中，表示对主语的尊敬（用于长辈、上级等）

<a id="h-특수"></a>
**特殊敬语词** —— 一些动词有专门的敬语形式，不是加-시-，而是整个词换掉：
먹다(吃)→드시다/잡수시다；자다(睡)→주무시다；있다(在)→계시다；말하다(说)→말씀하시다
