## 数据Schema（每条词汇JSON结构）

```json
{
  "korean": "...",
  "romanization": "...",
  "pos": "名词/动词/形容词/副词/依存名词/冠形词/感叹词/数词等",
  "meaning": "简洁中文释义（不含敬语括注）",
  "register": "中立/敬语/非敬语（반말）",
  "example": "原创韩语例句",
  "exampleTranslation": "中文翻译",
  "grammarBreakdown": [
    {"text":"...", "meaning":"...或—", "note":"语法说明", "isTarget": true（仅标在对应本词条headword的那一项上）}
  ],
  "sentenceType": "陈述句/疑问句/命令句（请求）/共动句",
  "formalityLevel": "格式体敬语（합니다体）/非格式体敬语（해요体）/非敬语（반말）",
  "homonymIndex": "01"（仅当该词形与其他已用词形拼写冲突时才有此字段，取自官方词典同形异义编号）
}
```

## 质量控制规则（已在全部40课严格执行）

1. **官方一致性**：每一课独立重新核对官方xlsx，逐字确认存在，0容忍编造
2. **已知例外**：以下"名词+하다"词条官方表只收录名词形式，不算错误：
   결혼하다/사랑하다/공부하다/빨래하다/산책하다/세수하다/요리하다/운동하다/일하다/청소하다
3. **同形异义词处理**：跨40课凡拼写相同、含义不同的词（如"이"=二/牙齿/这个），均已添加`homonymIndex`字段，取自官方词典编号。前端应在卡片正面用小号上标显示该编号。已知同形词清单见下方。
4. **isTarget标记**：全部1844条词条的grammarBreakdown数组中，均已标记出对应该卡片headword的那一项为`"isTarget": true`，前端应据此高亮显示。
5. **不规则动词/形容词说明**：凡例句中出现ㅂ/ㄷ/ㅅ/르/ㅎ不规则活用或으脱落、ㄹ脱落且实际改变了表面形态的，note字段中均已注明具体是哪类不规则、如何变化，不能只写"词干变化"这种模糊说法。
