## 项目是什么

一个韩语单词闪卡自学网站，个人项目，非商业用途，会发到小红书给别人免费用。
核心功能：翻卡片学词汇 + 印章式记忆分级（모르겠어요/조금 알아요/거의 알아요/완벽해요）
+ 间隔复习 + 语法分析 + 发音。

**只能用官方公开来源，绝对不能用某本具体教材（尤其是盗版渠道拿到的）的完整内容。**

- 唯一权威数据源：TOPIK官方词汇表 `토픽_어휘_목록_공개_목록.xlsx`
  （来源 topik.go.kr，元数据显示编辑者为 NIIED 韩国国立国际教育院）
  - `초급(1,847)` sheet：1,847个初级词，字段：수준/어휘/길잡이말/품사
  - `중급(3,873)` sheet：3,873个中级词
  - 官方没有独立的고급（高级）词表
- 每一个新词汇批次，**必须**逐词核对确实存在于这份官方xlsx里（用词形去掉数字后缀比对），
  不能凭印象/记忆生成，也不能假设"这本书里的词应该没问题"就直接用
- 已经发现并纠正过的错误：曾误把中级词（돌보다、사촌、일정）当初级词用，
  发现后换成了真正的初级词（안다、조카、시간표）——**说明必须每次都认真核对，不能想当然**
- 例句 100% 原创撰写，不引用任何教材/网站
- 有一份《完全掌握新韩国语能力测试TOPIK必备单词》(金龙一) 的盗版PDF/文字/RTF，
  用户多次尝试提供，**已明确拒绝使用**，不要用这本书的任何完整单元内容

## 数据结构（每条词汇的字段）

```json
{
  "korean": "국적",
  "romanization": "gukjeok",
  "pos": "名词",
  "meaning": "国籍",
  "register": "中立",
  "example": "제 국적은 중국입니다.",
  "exampleTranslation": "我的国籍是中国。",
  "grammarBreakdown": [
    {"text": "제", "meaning": "我的", "note": "저(我，敬语)+의(所有格助词)的缩略形式"},
    {"text": "국적", "meaning": "国籍", "note": "名词"},
    {"text": "은", "meaning": "—", "note": "主题助词，标记句子的主题"},
    {"text": "중국", "meaning": "中国", "note": "名词"},
    {"text": "입니다", "meaning": "是……", "note": "이다(是)的敬语终结形：词干+ -ㅂ니다"}
  ]
}
```

字段规则：
- `meaning`：只放纯词义，**不要**塞括号里的敬语/语体标注（那是 register 字段的事）
- `register`：敬阶信息单独放这里，值如"中立"、"敬语"、"非敬语（반말）"，
  只有不是"中立"才在卡片上显示小标签
- `pos`：显示时用【】包起来，如"【名词】"
- `grammarBreakdown`：逐词/逐助词拆解，每个元素是 text/meaning/note 三个字段，
  纯语法助词没有独立词义的，meaning 写"—"。默认折叠显示，点击展开成表格

## 文件交接工作流

- 用户在自己的对话里让另一个Claude实例生成新Unit的json文件，文件名格式
  `unitXX_주제韩语名.json`，XX是两位数编号
- 用户下载后会放进项目里的 `/incoming` 文件夹
- 看到 `/incoming` 里有新文件时：**完整替换**对应Unit的全部数据（所有字段），
  不要只更新部分字段（之前出过"只换了词，没加上词性"的问题，切记要全量替换）
- 替换后要抽查至少5-8个词，确认每个字段都真的显示出来了，不要只测第一条就当没问题
